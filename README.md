# Retail Sales Analysis Project

## Objective
This project analyzes retail sales data to identify sales trends, customer purchasing behavior, and business insights using Python.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Tasks Performed
- Loaded the retail sales dataset
- Performed data cleaning
- Analyzed sales trends
- Created visualizations
- Generated business insights

## Visualizations
- Sales Distribution
- Monthly Sales Trend
- Category-wise Sales
- Correlation Heatmap
